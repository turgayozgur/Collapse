Collapse
========

Collapse with C++

Nesneye yönelik yaklaşımla geliştirilen bir oyun yapmak amacıyla seçimlik projelerin arasından,
internette çok sık karşılaşabileceğiniz bir oyun yapmaya karar verdim.

O zamanlar daha görsel programlama hakkında bilgim olmadığı için C++ ile komut ekranı dediğimiz o siyah ekranda
sınırları zorlamaya çalıştım.Sonuç olarak ortaya gerek sesleri gerek görüntüsüyle,
ilginç ama eğlenceli bir proje çıktı:)

Nesneye yönelik yaklaşım, dört yönlü bağlı liste yapısı ve 
recursive mantığı projenin temellerini oluşturuyor.

**İyi Eğlencler! :)**
